说明

本页可直接用于论文表述：在外部自监督骨干下，共识拒识对高置信样本的准确率与聚类一致性具有稳定的平均提升。若95% CI跨0，则只能写平均提升，不能写稳定显著提升。

同时，ConvNeXt主方法所用本地检查点来源尚需在投稿前查清。因此当前不应把ConvNeXt与DINOv2的差距解释为骨干网络本身优劣，只能写为两个外部权重设置下的实验现象。
