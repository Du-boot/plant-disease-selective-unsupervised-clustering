五、结果解释

  • DINOv2+KMeans提供了强自监督视觉表示的外部无监督基线。它在F_new上较高，但在V_new、M_new和M_new_drop5_drop7上明显低于本文ConvNeXt主框架。
  • DINOv2+UMAP+all3相对DINOv2+KMeans提高了V和M数据集的保留样本准确率与ARI_kept，说明“三聚类对齐+共识拒识”并不完全依赖ConvNeXt这一单一骨干。
  • 在M_new_drop5_drop7上，DINOv2+UMAP+all3达到88.15%±0.73%，低于本文主方法90.72%±1.33%，但它支撑了外部自监督特征下共识拒识仍有效这一补充结论。
  • 选择性方法的acc_kept不能与全样本方法的acc_all直接等价排名，因此本页同时报告Coverage和保守全样本准确率。
