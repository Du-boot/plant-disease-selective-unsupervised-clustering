# 中文论文核心初稿（Protocol A 冻结版）

## 摘要（建议稿）

植物病害图像识别通常依赖大规模人工标注和有监督分类器训练，在新作物、新病害或标注资源不足场景下应用受限。本文提出一种基于预训练视觉特征和多算法共识的传导式选择性无监督聚类方法，用于植物叶片病害图像的高置信聚类识别。全部图像在不使用真实类别标签的条件下参与特征提取、降维、聚类、跨算法簇编号对齐和共识拒识；真实标签仅在聚类与拒识结果完全固定后用于多对一事后类别对齐和外部评价。方法首先利用ConvNeXt-XLarge结构和本地外部权重检查点提取2048维视觉特征，然后采用UMAP降至100维，并分别进行KMeans、Birch和Agglomerative聚类。以KMeans为参考聚类，通过样本重叠矩阵和Hungarian算法对齐不同聚类算法的簇编号，随后通过三算法一致性筛选高置信样本，不一致样本被拒识。

在统一的UMAP 100维和20簇配置下，该方法在水果病害子集、蔬菜病害子集和预先整理的多作物九类病害子集上分别获得99.95%±0.00%、97.85%±1.88%和90.72%±1.33%的保留样本事后对齐聚类准确率，对应拒识率分别为21.30%±4.10%、10.01%±3.88%和9.39%±4.51%。在蔬菜病害和多作物九类病害子集上，三算法严格共识相对于同覆盖率随机拒识分别提高1.66和2.88个百分点。结果表明，多算法共识能够在复杂病害数据集中筛选出更可靠的高置信样本。需要指出的是，本文评价的是传导式无监督聚类结果与真实病害类别的一致程度，而非对未参与聚类的新图像进行独立归纳分类。

## 方法关键段落

### 任务定义与评价协议

本文采用传导式选择性无监督聚类评价协议。给定一个待分析图像集合，全部图像在不使用真实类别标签的情况下共同参与视觉特征提取、低维流形构建、聚类、跨算法簇编号对齐和共识拒识。真实类别标签不参与任何模型拟合、降维、聚类、簇对齐或拒识决策，仅在所有聚类结果和拒识结果完全固定后，用于多对一事后类别映射和外部聚类评价。因此，本文报告的准确率不等同于独立测试分类器在新图像上的预测准确率。

### 数据审计

本文采用SHA256哈希识别并移除字节级完全重复图像。对于pHash检测出的高视觉相似图像，由于感知哈希无法可靠区分同源图像与具有相似病斑表型的不同叶片，因此未据此自动删除样本。pHash结果仅作为数据集近重复风险审计，不参与特征提取、降维、聚类、拒识或评价。

### 特征提取与降维

本文采用ConvNeXt-XLarge结构作为冻结视觉特征提取器。代码中使用`timm.create_model("convnext_xlarge_in22k", pretrained=False, num_classes=0, global_pool="avg")`初始化模型，并载入本地外部权重检查点`model.safetensors`。由于当前权重文件未包含可追溯的metadata，且尚未确认其最初训练来源，投稿前不将其直接表述为官方ImageNet-22K预训练权重。所有图像被调整为224×224，并采用均值[0.485, 0.456, 0.406]和标准差[0.229, 0.224, 0.225]进行归一化。模型权重在实验中保持冻结，最终输出2048维视觉特征。

2048维特征随后通过UMAP降至100维。主配置为`n_components=100`、`n_neighbors=15`、`min_dist=0.1`、`metric="euclidean"`，随机种子为11、22、33、44和55。为验证降维选择的影响，本文还在相同样本条件下比较了Raw 2048维、PCA 100维和UMAP 100维表示。

### 聚类、簇对齐与拒识

在低维特征空间中，本文分别采用KMeans、Birch和Agglomerative三种聚类算法。主配置中三种算法均设置为20个簇。KMeans使用`n_clusters=20`、`n_init=10`、默认`k-means++`初始化和最大迭代次数300；Birch使用`threshold=0.11`、`branching_factor=25`、`n_clusters=20`；Agglomerative使用`n_clusters=20`和Ward linkage，对应欧氏距离。

由于不同聚类算法的簇编号没有语义一致性，本文以KMeans为参考聚类，构建KMeans簇与另一算法簇之间的样本重叠矩阵。矩阵元素表示两个簇共同包含的样本数量。随后采用Hungarian算法最大化总样本重叠，得到Birch和Agglomerative相对于KMeans的簇编号对齐关系。若某个簇无法匹配，则标记为-1。簇编号对齐后，任意两个算法给出相同簇编号的样本定义为any2保留样本；三个算法全部给出相同簇编号的样本定义为all3保留样本；其余样本被拒识。

### 评价指标

本文将保留样本事后对齐聚类准确率记为：

```text
Coverage = N_kept / N

Acc_kept = N_correct,kept / N_kept

Conservative Overall Accuracy = N_correct,kept / N
                              = Acc_kept × Coverage
```

其中，`Acc_kept`表示保留样本在多对一事后类别映射后的聚类准确率；`Coverage`表示未被拒识的样本比例；保守全样本准确率将拒识样本计入全样本分母。该指标采用保守口径，并不表示被拒识样本实际上均被错误分类。

## 结果章节建议组织

### 主实验结果

在统一的UMAP 100维和20簇配置下，all3策略在F_new、V_new和M_new_drop5_drop7上分别获得99.95%±0.00%、97.85%±1.88%和90.72%±1.33%的保留样本事后对齐聚类准确率。对应拒识率分别为21.30%±4.10%、10.01%±3.88%和9.39%±4.51%，保守全样本准确率分别为78.65%±4.10%、88.01%±2.99%和82.14%±2.89%。G_new仅作为跨作物视觉类别辅助实验，不纳入三个病害主数据集的综合结论。

### 消融与拒识基线

完整消融比较包括KMeans、Birch、Agglomerative三个单算法基线，any2与all3两种共识拒识策略，KMeans距离拒识基线以及同覆盖率随机拒识基线。在F_new中，KMeans本身已经达到99.95%的保留样本事后对齐聚类准确率，all3没有进一步提升该指标，却将覆盖率降至78.70%，显示出明显天花板效应。在V_new和M_new_drop5_drop7中，all3相对于同覆盖率随机拒识分别提高1.66和2.88个百分点，说明严格共识在复杂数据集上具有更明确的选择性收益。

### 聚类结构指标

all3筛选后，保留子集的聚类结构指标相对于全部样本有所改善。F_new的ARI由0.389提高到0.517，V_new由0.691提高到0.764，M_new_drop5_drop7由0.522提高到0.578。这表明共识拒识不仅改变了保留样本的类别准确率，也提高了保留子集的聚类结构一致性。然而，ARI并未接近1，说明20簇设置下同一病害类别可能被拆分为多个高纯度视觉子簇。

### 公平降维对照

在相同样本条件下比较Raw 2048维、PCA 100维和UMAP 100维表示后可以看到，UMAP并非在所有数据集和所有指标上全面最优。其主要优势体现在V_new和M_new_drop5_drop7等类别结构更复杂的数据集中，能够改善覆盖率与保守全样本准确率；在F_new等结构清晰数据集中，原始2048维特征在部分指标上并不弱于UMAP。因此，UMAP 100维应表述为跨数据集统一主配置，而不是逐数据集最优参数。

### Bootstrap统计

种子级配对Bootstrap结果显示，V_new中all3相对于同覆盖率随机拒识的`Acc_kept`提高1.66个百分点，95% CI为[0.58%, 3.15%]；M_new_drop5_drop7中相应提升为2.88个百分点，95% CI为[1.65%, 4.11%]。M_new_drop5_drop7中all3相对于KMeans距离拒识提高1.47个百分点，95% CI为[0.87%, 2.06%]。V_new中all3相对于KMeans距离拒识的均值更高，但95% CI为[-0.19%, 2.40%]，跨过0，因此不能声称其稳定显著优于距离拒识。

### 类别级分析

类别级结果显示，M_new_drop5_drop7中的Tomato–Verticillium wilt为最薄弱类别，保留样本事后对齐聚类准确率为66.12%±6.85%，拒识率为26.24%±15.78%。Rice–brown spot和Corn–streak virus的准确率也相对较低，分别约为83.14%和83.71%。V_new中的Tomato–late blight跨种子波动较大。上述结果说明，总体高置信准确率可能掩盖少数类别的持续混淆，类别级拒识率和混淆矩阵应作为解释拒识机制的重要补充。

## 讨论关键段落

### F_new的天花板效应

F_new中单独KMeans已经获得接近饱和的事后对齐聚类准确率，严格三算法一致不会进一步提高`Acc_kept`，反而带来明显覆盖率损失。因此，F_new不能作为all3提升准确率的主要证据，而应作为基础聚类结构清晰时严格拒识可能过于保守的例子。

### V_new与M_new_drop5_drop7的核心证据

V_new和M_new_drop5_drop7更能体现多算法共识的价值。在这些类别结构更复杂的数据集中，all3相对于同覆盖率随机拒识取得更高的保留样本事后对齐聚类准确率，说明跨算法一致性能够比随机删除更有效地筛选高风险样本。特别是在M_new_drop5_drop7中，all3相对于KMeans、any2、KMeans距离拒识和随机拒识均显示出稳定提升，是支撑本文方法创新性的核心证据。

### 高`Acc_kept`与ARI较低的关系

本文采用20个聚类簇，而真实病害类别数通常少于20。因此，同一病害类别可能被拆分为多个视觉子簇。多对一事后类别映射后，这些高纯度子簇可以共同映射到同一病害类别，从而获得较高`Acc_kept`；但从整体聚类结构看，病害类别与簇并非一一对应，因此ARI不会接近1。这说明本文方法更倾向于发现类内视觉子结构，而非严格恢复“一种病害对应一个簇”的结构。

### 方法边界

本文评价的是传导式无监督聚类结果与真实类别之间的一致性，而不是训练一个能够直接预测未参与聚类新图像的归纳式分类器。未来工作需要进一步研究如何将当前传导式聚类框架扩展为可增量更新或可归纳预测的新样本识别系统。

## M数据集写法

本文同时报告完整11类`M_new`和预先整理的9类子集`M_new_drop5_drop7`。9类子集未纳入`Potato - nematode`和`Rice - leaf blast`两个类别。其中，`Potato - nematode`样本量明显较少，且属于线虫危害，与典型叶部真菌、细菌或病毒病斑表型存在任务范围差异。对于`Rice - leaf blast`，若无额外质量异常证据，不将其描述为错误或低质量类别，而仅将9类版本作为预先整理的子集分析。完整11类结果同步报告，以避免选择性删类质疑。

## 局限性

本文仍存在若干局限。第一，当前评价协议为传导式无监督聚类评价，尚未证明模型对完全未参与聚类的新图像具有归纳式预测能力。第二，视觉特征提取依赖单一ConvNeXt-XLarge结构和本地外部权重检查点，权重最初来源仍需在投稿前补充确认。第三，公开数据集中的图像采集条件相对受控，真实田间复杂背景下的泛化能力仍需进一步验证。第四，多对一事后映射能够评价聚类结果与真实类别的一致性，但也意味着高`Acc_kept`不等同于一类一簇的聚类结构恢复。
